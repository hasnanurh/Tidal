How to install on your Android:

1. Extract the zip and put the folder on a web server 
    — the easiest free option is GitHub Pages (free hosting, just push the folder to a repo and enable Pages in settings)
2. Open the URL in Chrome on your tablet
3. Chrome will show an "Add to Home Screen" banner — tap it
4. TIDAL installs like a native app, opens in landscape fullscreen, and works offline
